Attributes of <b:modal >
Attribute Default value Description
backdrop true By default, you can close a modal dialog by clicking somewhere outside the modal. Set backdrop="false" to disable this feature.
closeOnEscape true By default, users can close modal dialogs by hitting the ESC key. Set close-on-escape="false" to disable this feature.
id (none) Unique identifier of the component in a namingContainer.
rendered false Boolean value to specify the rendering of the component, when set to false component will not be rendered.
rendered false Boolean value to specify the rendering of the component, when set to false component will not be rendered.
size (none) Modal's size. Possible values modal-sm, modal-lg
styleClass (none) Style class of this element.
title (none) Bold title displayed in Modal's header.
